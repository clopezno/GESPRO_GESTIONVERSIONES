TourBloc

TourBloc class

TourBloc es el Bloc encargado de gestionar el estado de los tours.

Gestiona la lógica relacionada con la creación, modificación, y visualización de tours turísticos, además de interactuar con servicios como Gemini, Google Places y un servicio de optimización de rutas.
